## Arguments

<ParamField body="linkIds" type="string[]" required>
  Comma-separated list of link IDs to delete. Maximum of 100 IDs. Non-existing
  IDs will be ignored.
</ParamField>

## Response

<ResponseField name="deletedCount" type="number" required>
  The number of links deleted.
</ResponseField>

<RequestExample>

```ts
await dub.links.deleteMany({
  linkIds: ["clux0rgak00011...", "clux0rgak00022..."],
});
```

</RequestExample>

<ResponseExample>

```ts
{
  deletedCount: 2;
}
```

</ResponseExample>
